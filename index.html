Fitness Management Platform

GMG Fitness Studio App

Multi-role gym SaaS with separate portals for the owner, each trainer, and every member. React 18, Firebase Auth, and Firestore powering live data across all three roles.

React 18 Firebase Auth Firestore Tailwind React Router
  • Three portals, one codebase. Owner sees revenue, trainer count, and member roster. Trainer gets their client list and daily sessions. Member books slots and tracks workouts.
  • Smart conflict checks. The system rejects bookings if a trainer is already occupied in that slot — no manual checking required.
  • Billing and inventory tracked in Firestore with paginated reads and role-scoped security rules.

DairyFlow Delivery Manager

Mobile-first field app for dairy delivery agents. Drag a card right to log a delivery, adjust litres on the spot, and the app generates a complete WhatsApp billing message automatically.

React 19 Vite Framer Motion Firebase WhatsApp API
  • Swipe to record. Drag card right = delivered. Left = failed. Any phone, zero training needed.
  • Auto billing on month end. Total litres × rate = bill. WhatsApp message with UPI link generated and ready to send.
  • Owner dashboard sees route completion, per-agent stats, and collection summaries in real time.
